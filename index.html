    Dicas para comprimir melhor

    Quer reduzir mais?

    Diminua a qualidade das imagens. Esse é o ajuste que mais influencia o tamanho final.

    Passadas ajudam, mas têm limite

    Mais passadas podem reduzir um pouco mais, porém demoram mais e o ganho costuma diminuir.

    PDFs escaneados reduzem mais

    Arquivos com muitas imagens ou páginas escaneadas costumam ter maior redução.

    PDFs só com texto reduzem menos

    Se o PDF já for leve ou tiver pouco conteúdo de imagem, a compressão pode ser pequena.

    Muitos PDFs podem pesar

    Para juntar e comprimir vários PDFs, prefira começar com até 10 arquivos ou até 200 MB no total.

    Confira o resultado final

    Depois de baixar, abra o PDF final e veja se a qualidade ficou adequada.

    Por que o tamanho final pode variar?

    O tamanho final depende da estrutura interna do PDF. Dois compressores podem gerar resultados diferentes mesmo usando níveis parecidos de qualidade, porque cada ferramenta pode tratar imagens, fontes, metadados e objetos internos de forma diferente.

    No Comprimir PDF, o processamento acontece diretamente no navegador, priorizando privacidade. Ferramentas online que processam arquivos em servidores podem usar otimizações adicionais. Ainda assim, ao reduzir mais a qualidade das imagens, é possível alcançar compressões semelhantes em muitos PDFs.

    A maior diferença normalmente vem da qualidade e da resolução das imagens internas.

    Como verificar?

    Você pode verificar que os PDFs não são enviados para servidores usando as ferramentas de desenvolvedor do navegador.

    1. Abra as Ferramentas de Desenvolvedor.
    2. Acesse a aba Rede ou Network.
    3. Selecione os PDFs.
    4. Inicie a compressão ou junção.
    5. Verifique que não há upload dos arquivos PDF.
    6. O processamento ocorre localmente por JavaScript no navegador.

    Se a aplicação carregar arquivos externos, como bibliotecas, fontes ou a imagem do QR Code Pix, essas requisições podem aparecer na aba Rede/Network. Isso não significa que seus PDFs foram enviados. O que deve ser verificado é que os arquivos PDF selecionados não aparecem como upload ou envio para servidor.

    Política de Privacidade — Comprimir PDF

    O Comprimir PDF foi criado para comprimir e juntar arquivos PDF diretamente no navegador do usuário, sem necessidade de envio dos documentos para servidores.

    1. Processamento local

    Os arquivos PDF selecionados pelo usuário são processados localmente no navegador. Os documentos não são enviados para servidores, não são armazenados em banco de dados, não são copiados para sistemas externos e não são compartilhados com terceiros.

    2. Conteúdo dos arquivos

    O Comprimir PDF não acessa, analisa, armazena ou transmite o conteúdo dos documentos selecionados. O processamento ocorre na memória local do navegador, com a finalidade técnica de comprimir, organizar, juntar e gerar uma nova versão dos PDFs escolhidos pelo usuário.

    3. Dados coletados

    O Comprimir PDF não coleta o conteúdo dos arquivos PDF. Caso a aplicação utilize estatísticas técnicas de acesso, elas devem ser limitadas a informações gerais, como data de acesso, tipo de navegador, páginas acessadas e dados necessários para funcionamento e segurança do site, sem incluir conteúdo, nome dos arquivos ou dados internos dos documentos.

    4. Recursos externos

    A aplicação pode carregar recursos externos necessários à interface, como a imagem do QR Code Pix hospedada em serviço de imagem. Essas requisições externas não devem incluir o conteúdo dos PDFs selecionados pelo usuário.

    5. Cookies e preferências locais

    Se cookies ou localStorage forem utilizados, eles devem servir apenas para preferências locais, como tema claro ou escuro, modo escolhido ou configurações de interface. Cookies de publicidade, rastreamento ou análise comportamental não devem ser adicionados sem aviso claro ao usuário e sem opções adequadas de consentimento.

    6. Responsabilidade do usuário

    O usuário é responsável pelos arquivos que seleciona para compressão ou junção. Ao utilizar o Comprimir PDF, o usuário declara possuir autorização para processar, comprimir, reorganizar, juntar ou gerar cópia dos documentos escolhidos.

    7. Segurança

    Como o processamento ocorre localmente, o risco de exposição dos documentos é reduzido. Ainda assim, recomenda-se que o usuário utilize a ferramenta em dispositivo confiável, navegador atualizado e conexão segura.

    8. Verificação técnica

    O usuário pode verificar que os PDFs não são enviados para servidores utilizando as ferramentas de desenvolvedor do navegador, especialmente a aba Rede/Network, durante o processo.

    9. Doações via Pix

Antes de confirmar qualquer pagamento, o usuário deve conferir se o recebedor aparece como Marcos Guimarães Duailibi Filho e se os dados do Pix correspondem aos exibidos no site.

    Termos de Uso — Comprimir PDF

    2. Processamento local

    A compressão e a junção são realizadas localmente no navegador. Os arquivos selecionados não são enviados para servidores, não são armazenados pela aplicação e não são compartilhados com terceiros.

    Transparência técnica

    O Comprimir PDF funciona com HTML, CSS e JavaScript executados no navegador. Quando o usuário escolhe os PDFs, os arquivos são carregados na memória local do navegador para que a compressão e a junção sejam realizadas.

    A aplicação não precisa enviar os documentos para servidor para executar o processamento. O arquivo final é gerado no próprio navegador e disponibilizado para download local.

    O processamento pesado roda em um Web Worker (thread separada) para não travar a interface. Usuários avançados podem verificar esse funcionamento pelo código-fonte e pela aba Rede/Network das ferramentas de desenvolvedor.
