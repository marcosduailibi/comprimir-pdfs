# demo/

> Esta pasta é apenas referência. A aplicação principal **Comprimir PDF** agora
> está em [`../index.html`](../index.html).

Para rodar localmente:

```bash
cd libjs/pdfcompress
npm run start        # sobe um servidor estático na raiz do pacote
```

Abra **http://localhost:3000/** (o `index.html` na raiz de `pdfcompress`).

O app anterior (somente compressão) foi substituído pela versão completa
(comprimir + juntar + Web Worker + Pix).
